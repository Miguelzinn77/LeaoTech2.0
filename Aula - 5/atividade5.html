O Leão Tech é um curso voltado para o desenvolveminto de habilidade em tecnologia,
incentivando os alunos a aprender HTML, CSS e JavaScript de forma prática.



Tema

Desenvolvimento Web

Público

Alunos do Leão Tech

Foco

HTML, CSS e JavaScript


O curso é projetado para ser acessível a todos, independentemente do nível de experiência, e tem como objetivo capacitar os alunos a criar seus próprios projetos web.
